🔀 JavaScript if / else Summary

📌 Overview

The if statement executes a block of code if a condition is true. Optionally, else or else if can be used to handle other cases.

if (condition) {
  // code to execute if condition is true
} else if (anotherCondition) {
  // code if anotherCondition is true
} else {
  // code if none of the above conditions are true
}

⚠️ Condition Conversion

Conditions are evaluated using boolean coercion. Some falsy values include:

false
0
"" (empty string)
null
undefined
NaN

All other values are considered truthy.

🧪 Ternary Operator

A shorter form of if...else for assignments or simple logic:

let result = condition ? value1 : value2;

It can be nested but avoid overusing as it reduces readability.

👨‍💼 Senior/Lead Considerations

  • Favor early returns in functions to reduce nested if blocks.
  • For complex branching, consider using switch or a map object for clarity.
  • Keep logic declarative and readable; avoid deeply nested conditions.
  • Use ternary only when expressions are short and readable.
  • In critical systems, always consider implications of falsy values like 0 or empty strings.
