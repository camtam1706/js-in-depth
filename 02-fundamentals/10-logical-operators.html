üü¢ Logical OR (||)

The OR operator returns the first truthy value among its operands. If all operands are falsy, it returns the last operand. This behavior is known as short-circuit evaluation. 


let result = value1 || value2 || value3;

Example: 


let name = "";
let nickname = "Coder";
let displayName = name || nickname || "Anonymous";
console.log(displayName); // Output: "Coder"

üî¥ Logical AND (&&)

The AND operator returns the first falsy value among its operands. If all operands are truthy, it returns the last operand. This also utilizes short-circuit evaluation. 


let result = value1 && value2 && value3;

Example: 


let isLoggedIn = true;
let hasPermission = false;
let access = isLoggedIn && hasPermission;
console.log(access); // Output: false

‚ö™ Logical NOT (!)

The NOT operator inverts the truthiness of its operand. It converts the operand to a boolean and returns the opposite value. 


let result = !value;

Example: 


let isAvailable = false;
console.log(!isAvailable); // Output: true

üü° Nullish Coalescing (??)

The nullish coalescing operator returns the right-hand operand when the left-hand operand is null or undefined. 


let result = value1 ?? value2;

Example: 


let userInput = null;
let defaultValue = "Default";
let result = userInput ?? defaultValue;
console.log(result); // Output: "Default"
