üí° What is the Nullish Coalescing Operator?

The ?? operator returns the first operand if it's not null or undefined; otherwise, it returns the second operand. 

let user = null;
let name = user ?? "Anonymous"; // "Anonymous"

üîç Difference Between ?? and ||

The || operator returns the first truthy value, which means it considers 0, "", and false as falsy. In contrast, ?? only considers null and undefined as nullish. 

let height = 0;
console.log(height || 100); // 100
console.log(height ?? 100); // 0

üß™ Common Use Cases

  • Providing default values when variables are null or undefined.
  • Chaining multiple values to find the first defined one.
  • Combining with optional chaining (?.) for safe property access.
let firstName = null;
let lastName = null;
let nickName = "Supercoder";
let name = firstName ?? lastName ?? nickName ?? "Anonymous"; // "Supercoder"

‚ö†Ô∏è Important Considerations

  • Operator Precedence: ?? has lower precedence than most operators, so use parentheses when combining with other operators.
  • Syntax Restrictions: Cannot be used directly with || or && without parentheses.
// Correct usage with parentheses
let result = (a && b) ?? c;

üß† Notes for Senior/Lead Developers

  • Code Clarity: Ensure team members understand the difference between ?? and || to prevent unintended behavior.
  • Browser Compatibility: ?? is part of ES2020; ensure that target environments support it or include necessary polyfills.
  • Code Reviews: Watch for misuse of ?? where || might be more appropriate, especially when dealing with falsy values like 0 or "".
