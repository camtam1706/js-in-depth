The while loop executes a block of code as long as the specified condition is true.

let i = 0;
while (i < 3) {
  alert(i);
  i++;
}

ğŸ” do...while Loop

Executes the body first, then checks the condition:

let i = 0;
do {
  alert(i);
  i++;
} while (i < 3);

ğŸ”‚ for Loop

For concise iteration when loop structure is predictable:

  alert(i);
}
