üìå What is a Function?

Functions are reusable blocks of code that perform a specific task. They help in organizing code, avoiding repetition, and improving readability.

üõ†Ô∏è Function Declaration

Defines a function with the function keyword, followed by a name, parameters, and a body.

function greet() {
  alert('Hello!');
}

Function declarations are hoisted, meaning they can be called before they are defined in the code.

üßæ Function Expression

Defines a function as part of an expression, often assigned to a variable.

const greet = function() {
  alert('Hello!');
};

Function expressions are not hoisted; they can only be called after they are defined.

üîÅ Parameters and Arguments

Functions can accept inputs called parameters, which are specified during function definition. When calling the function, actual values called arguments are passed.

function greet(name) {
  alert('Hello, ' + name);
}

greet('Alice');

üì¶ Return Values

Functions can return values using the return statement. This allows the function to output a result that can be used elsewhere in the code.

function add(a, b) {
  return a + b;
}

let sum = add(2, 3); // sum is 5

üîí Scope and Variables

Variables declared inside a function are local to that function and cannot be accessed outside. Functions can also access variables from their outer scope.

let user = 'Alice';

function greet() {
  let message = 'Hello, ' + user;
  alert(message);
}
