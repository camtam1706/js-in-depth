Avoid using comments to explain what the code does. Instead, write self-explanatory code. For example:

// This code checks if i is a prime number
for (let j = 2; j < i; j++) {
  if (i % j == 0) continue nextPrime;
}

Can be refactored to:

function isPrime(n) {
  for (let i = 2; i < n; i++) {
    if (n % i == 0) return false;
  }
  return true;
}
