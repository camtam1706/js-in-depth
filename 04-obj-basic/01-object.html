üîë Object Basics

Objects store keyed collections of various data and more complex entities.

// Creating an object using object literal
let user = {
  name: "John",
  age: 30
};

// Accessing properties
console.log(user.name); // John
console.log(user["age"]); // 30

// Adding a new property
user.isAdmin = true;

// Deleting a property
delete user.age;

üß† Object Methods and this

Functions assigned to object properties are called methods. Inside a method, this refers to the object the method was called on.

let user = {
  name: "John",
  sayHi() {
    console.log(this.name);
  }
};

user.sayHi(); // John

Arrow functions do not have their own this and inherit it from the parent scope.

let user = {
  name: "John",
  sayHi() {
    let arrow = () => console.log(this.name);
    arrow();
  }
};

user.sayHi(); // John

üõ†Ô∏è Useful Object Methods

JavaScript provides several methods to work with objects:

  • Object.keys(obj): Returns an array of the object's keys.
  • Object.values(obj): Returns an array of the object's values.
  • Object.entries(obj): Returns an array of [key, value] pairs.
  • Object.fromEntries(arr): Converts an array of [key, value] pairs back into an object.
let user = {
  name: "John",
  age: 30
};

console.log(Object.keys(user)); // ["name", "age"]
console.log(Object.values(user)); // ["John", 30]
console.log(Object.entries(user)); // [["name", "John"], ["age", 30]]

üöÄ Advanced Concepts

Objects can be copied and merged using Object.assign:

let user = { name: "John" };
let permissions = { canView: true };

let userWithPermissions = Object.assign({}, user, permissions);
console.log(userWithPermissions); // { name: "John", canView: true }

To create a deep clone of an object, use structuredClone (modern browsers) or a custom function.

let user = {
  name: "John",
  sizes: {
    height: 182,
    width: 50
  }
};

let clone = structuredClone(user);
console.log(clone.sizes === user.sizes); // false
