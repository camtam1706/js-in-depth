ğŸ”„ The this Keyword

The this keyword refers to the current execution context and varies based on invocation:

  • In object method calls, this points to the owning object.
  • Standalone function calls default this to undefined in strict mode (or global object otherwise).
  • Arrow functions inherit this from their lexical (surrounding) scope.
  • Using call, apply, or bind allows explicit this binding. 
// As object method
const obj = {
  x: 10,
  getX() { return this.x; }
};
console.log(obj.getX()); // 10

// Arrow function inheriting this
function Counter() {
  this.count = 0;
  setInterval(() => {
    this.count++;
    console.log(this.count);
  }, 1000);
}
new Counter();

// Explicit binding
function greet() { return this.name; }
const user = { name: 'Alice' };
console.log(greet.call(user)); // 'Alice'
