ğŸ“˜ What is an Iterable?

An Iterable is any object that implements a [Symbol.iterator] method, allowing it to be used with for...of, spread syntax, or Array.from().

let str = "hello";
for (let char of str) {
  console.log(char); // h, e, l, l, o
}

ğŸ§­ What is an Iterator?

An Iterator is an object with a next() method that returns { value, done } pairs.

let range = {
  from: 1,
  to: 3,

  [Symbol.iterator]() {
    return {
      current: this.from,
      last: this.to,
      next() {
        if (this.current <= this.last) {
          return { done: false, value: this.current++ };
        } else {
          return { done: true };
        }
      }
    };
  }
};

for (let num of range) {
  console.log(num); // 1, 2, 3
}

ğŸ§° Senior/Lead Engineer Notes

  • When to create your own iterable: For custom data structures or lazy evaluation.
  • Best Practices: Always check for [Symbol.iterator] in third-party APIs.
  • Prefer for...of for readability over manual next().
  • Use Array.from(obj) for safe iterable to array conversion.
  • Avoid overcomplicating iterator logic. Keep it readable and testable.
