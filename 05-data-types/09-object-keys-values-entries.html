🧩 Object.keys, values, entries

🔑 Overview

The methods Object.keys(obj), Object.values(obj), and Object.entries(obj) are used to retrieve an array of keys, values, or [key, value] pairs from an object, respectively.

⚙️ Usage

These methods are particularly useful for iterating over object properties or transforming objects. For example: 

let user = { name: "John", age: 30 };

Object.keys(user); // ["name", "age"]
Object.values(user); // ["John", 30]
Object.entries(user); // [["name", "John"], ["age", 30]]

🔄 Transforming Objects

To transform an object, you can use Object.entries() to convert it into an array, apply array methods like map(), and then convert it back to an object using Object.fromEntries(). 

let prices = { banana: 1, orange: 2, meat: 4 };

let doublePrices = Object.fromEntries(
  Object.entries(prices).map(([key, value]) => [key, value * 2])
);

// doublePrices = { banana: 2, orange: 4, meat: 8 }

🧠 Notes

  • Object.keys, Object.values, and Object.entries ignore properties keyed by symbols.
  • To include symbol keys, use Object.getOwnPropertySymbols(obj) or Reflect.ownKeys(obj).
  • These methods return real arrays, allowing the use of array methods like map, filter, etc.

📌 For Senior/Lead Developers

  • Be cautious when using these methods on objects with a large number of properties, as they create arrays which can impact performance.
  • Remember that the order of properties is not guaranteed in objects; if order matters, consider using Map.
  • When transforming objects, ensure that the transformations are pure functions to maintain immutability and avoid side effects.
