The global object provides variables and functions that are available anywhere. By default, those that are built into the language or the environment.   • Variables declared with var in the global scope become properties of the global object.

  • Functions declared in the global scope become methods of the global object.
  • Using global variables is generally discouraged. There should be as few global variables as possible.
