🧠 Arrays in JavaScript

📌 Introduction to Arrays

Arrays are special data types in JavaScript that allow storing multiple values in a single variable, accessed by zero-based index.

📥 Declaring Arrays

const arr1 = [];
const arr2 = new Array();
const arr3 = [1, 2, 3];
const arr4 = new Array(1, 2, 3);
const arr5 = new Array(5); // [undefined × 5]

⚠️ Prefer [] over new Array() to avoid confusion with single numeric arguments.

🔧 Accessing & Mutating Elements

const fruits = ["Apple", "Banana", "Orange"];
console.log(fruits[0]); // "Apple"
fruits[1] = "Mango"; // Updates "Banana" to "Mango"

🔐 const vs let

const arr = [1, 2, 3];
arr[0] = 10; // ✅ OK
arr = [4, 5, 6]; // ❌ Error

let arr2 = [1, 2, 3];
arr2 = [4, 5, 6]; // ✅ OK

🛠️ Mutating Methods

  • push(): Adds to end 
    let a = [1];
a.push(2); // [1, 2]
  • pop(): Removes from end 
    let a = [1, 2];
a.pop(); // [1]
  • shift(): Removes from start 
    let a = [1, 2];
a.shift(); // [2]
  • unshift(): Adds to start 
    let a = [2];
a.unshift(1); // [1, 2]
  • splice(): Add/Remove at index 
    let a = [1, 2, 3];
a.splice(1, 1); // [1, 3]

🧪 Non-Mutating Methods

  • slice(): Extracts elements 
    let a = [1, 2, 3];
a.slice(1); // [2, 3]
  • map(): Transforms each element 
    [1, 2].map(x => x * 2); // [2, 4]
  • filter(): Filters elements 
    [1, 2, 3].filter(x => x > 1); // [2, 3]
  • reduce(): Reduces to single value 
    [1, 2, 3].reduce((a, b) => a + b, 0); // 6

🧬 Copying Arrays

const original = [1, 2, 3];
const copy1 = original.slice();
const copy2 = [...original];

✅ Use slice() or spread operator to safely clone arrays.

🧭 Stack vs Heap

JavaScript stores primitive types in the stack and reference types (like arrays) in the heap.

let x = 10;      // stored in stack
let y = x;       // copy value

let arr1 = [1];  // arr1 points to heap
let arr2 = arr1; // arr2 points to same heap
arr2[0] = 2;
console.log(arr1); // [2]
