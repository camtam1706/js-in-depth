What is JavaScript?

Dynamic Language

JavaScript is dynamic, interpreted (JIT compiled), runs on browser and server, and has no static types.

Async & Single-threaded

JS uses Event Loop + Web APIs to handle async operations in a single-threaded environment.

Multi-paradigm

Supports both OOP and FP, combining flexibility with structure for scalable architecture.

JavaScript Engines

Each browser has its own engine (V8, SpiderMonkey...) and JS is compiled JIT for performance.

JS Runtime

Consists of JS engine and browser APIs. Understand the execution model to debug effectively.

DOM & Web APIs

JS talks to DOM via Web APIs. DOM is not part of the language but provided by browser environment.
