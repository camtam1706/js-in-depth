üìä Operator Precedence

Operators with higher precedence are evaluated first. Associativity determines the direction of evaluation.

üîé Optional Chaining Playground: ?.

Optional chaining allows safe access to deeply nested properties. It short-circuits and returns undefined instead of throwing an error.

Expression:

const email = user?.profile?.email

Result:

...

What happens?

  • If user.profile doesn't exist: the expression returns undefined, no error.
  • If user.profile is null: still safe, returns undefined.
  • If profile exists and has email: it returns the email normally.
