üß† Memory Considerations

JS distinguishes between:

  • Primitive types: copied by value
  • Reference types: copied by reference (objects, arrays)

Changes to a reference affect all variables pointing to it!

üß¨ Prototype & Inheritance

Prototypal chain example:

const mammal = { eats: true };
const dog = Object.create(mammal);
console.log(dog.eats); // true

Objects inherit from other objects via prototype chain.

üß™ Advanced Primitives

Use Symbol.toPrimitive to customize object conversion:

const obj = {
  [Symbol.toPrimitive](hint) {
    return hint === 'string' ? 'custom' : 42;
  }
};
console.log(`${obj}`); // "custom"
console.log(+obj);     // 42
