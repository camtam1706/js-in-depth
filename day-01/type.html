ðŸ§© Object vs Array

In JavaScript, both Arrays and Objects are of type object. However, they are used differently and can be distinguished using specific methods.

Aspect Object Array
Usage Key-value pairs Ordered collection
Detection typeof obj === 'object' Array.isArray(arr)
Iteration for...in for, forEach, map 
const arr = [1, 2, 3];
const obj = { a: 1, b: 2 };
console.log(typeof arr); // 'object'
console.log(Array.isArray(arr)); // true

ðŸ§  Function vs Method

Technically, all methods are functions. The difference is how they're called and whether they have an associated context (this).

Aspect Function Method
Context Global or undefined Bound to object
Syntax functionName() obj.methodName()
Access to this No Yes 
function sayHi() {
  console.log('Hi');
}

const person = {
  name: 'Alice',
  greet() {
    console.log('Hello, ' + this.name);
  }
};

sayHi();         // Function
person.greet();  // Method
