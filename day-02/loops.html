🔁 JavaScript Loops Overview

1. 🌀 for Loop

Use when you know how many times to iterate:

for (let i = 0; i < 5; i++) {
  console.log(i);
}

2. 🔁 while Loop

Runs as long as the condition is true:

let i = 0;
while (i < 5) {
  console.log(i);
  i++;
}

3. 🔂 do...while Loop

Executes at least once before checking condition:

let i = 0;
do {
  console.log(i);
  i++;
} while (i < 5);

4. 🧵 for...of Loop (ES6)

Used to iterate over arrays, strings, sets, etc.:

for (const char of "hello") {
  console.log(char);
}

5. 🗂 for...in Loop

Used to iterate over object properties (not for arrays!):

const user = { name: "Alice", age: 30 };
for (const key in user) {
  console.log(key + ':', user[key]);
}

6. ⛔ break & continue

break exits the loop, continue skips current iteration:

for (let i = 0; i < 5; i++) {
  if (i === 3) break;
  console.log(i);
}
for (let i = 0; i < 5; i++) {
  if (i === 2) continue;
  console.log(i);
}

7. 🧠 Best Practices for Senior/Tech Leads

  • Use for...of with arrays/iterables, not for...in
  • Avoid modifying array/object during iteration (use pure functions)
  • Know when to break early for performance
  • Write clean, intention-revealing loops (avoid nested loops when possible)
  • Replace loops with higher-order functions (map, reduce) when readability improves
