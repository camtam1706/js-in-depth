üåê What is Scope?

Scope determines the accessibility of variables. JavaScript uses lexical scoping, meaning scope is determined by where code is written, not where it is called.

üß≠ Types of Scope

  • Global Scope: Accessible from anywhere in the code.
  • Function Scope: Defined inside a function, accessible only within that function.
  • Block Scope: Variables declared with let or const are scoped to the nearest block.
  • Lexical Scope: Inner functions access variables from their outer function's scope.

üìò Code Example

const x = 10;

function outer() {
  const y = 20;

  function inner() {
    const z = 30;
    console.log(x, y, z); // 10, 20, 30
  }

  inner();
}

outer();
